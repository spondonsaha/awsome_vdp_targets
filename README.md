# 🌐 Vulnerability Disclosure Programs List

A curated list of **Vulnerability Disclosure Programs (VDP)** and **Bug Bounty Programs** from companies, universities, and government organizations.

This repository helps **security researchers and bug bounty hunters** find places to responsibly report vulnerabilities.

⚠️ Always read the **scope and rules** of each program before testing.

Feel free to **add new VDP or bug bounty programs** by submitting a pull request.
